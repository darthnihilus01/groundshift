# 🪐 ExoDetect: AI-Powered Exoplanet Detection from Light Curves

ExoDetect is an open-source project that uses machine learning and deep learning to identify exoplanets from stellar light curve data. By analyzing tiny periodic dips in a star’s brightness, the system learns to distinguish real planetary transits from noise and false signals.

Built using real astronomical datasets (NASA Kepler/TESS), this project replicates the core methodology used in modern space missions to discover planets beyond our solar system.

## 🚀 Features

* 📊 Light curve visualization and preprocessing
* 🔍 Transit detection (dip identification)
* 🤖 AI-based classification (planet vs non-planet)
* 🧠 Deep learning models for time-series analysis (1D CNN)
* 📈 Confidence scoring and prediction outputs
* 🌌 Support for real-world datasets (Kepler / TESS)

## 🧠 Tech Stack

* Python
* NumPy, Pandas
* Matplotlib / Plotly
* PyTorch / TensorFlow

## 🎯 Goal

To bridge astronomy and AI by creating an accessible, research-inspired tool for detecting exoplanets using real scientific data.

## 🌍 Why This Matters

Exoplanet discovery is one of the most exciting frontiers in modern science. This project demonstrates how AI can accelerate scientific discovery and make advanced space research more accessible to developers and students.
